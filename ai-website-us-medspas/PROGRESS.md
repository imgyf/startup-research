## Verdict

**Score 5.0/10 — CONDITIONAL.** The pivot fixes Singapore's two real problems (low ACV ~USD 1k; PSG subsidy at 70% matching the unsubsidized price) but introduces two new ones of comparable weight: a **saturated, mature US med spa web/marketing agency market** (10+ specialists + huge GoHighLevel white-label long tail + Moxie eating the warmest leads + "free demo" already standard) and **offshore-operational friction** (US LLC, EIN 4–8 wk, Form 5472 $25k-penalty risk, banking, tax opinion, 13–14h timezone tax). Per-deal economics are *better* ($3.6–6.4k gross profit/build, 60–90% margin) but competitive advantage and founder-market fit are *worse* than the SG version. Net: roughly a lateral move, not the upgrade you'd want before committing.

**Recommendation:** Run the ~$1,000 / ~30-day validation pilot (`06-validation/validation-playbook.md`) BEFORE forming a US LLC. Deciding tests: Exp 2 (spec-site pitch beats generic pitch in a SMMA-saturated vertical?) and Exp 3 (US med spas hire an unknown offshore micro-team?). If both pass → pursue (form Wyoming LLC, scale Tampa+Orlando). If Exp 2 fails → stop pivot-shopping; "AI website agency" is structurally weak for this team; go to a fresh brainstorm. If forced to choose now without the pilot → marginally prefer staying with SG-restaurants (lower friction, validation path already mapped). Best of all → validate before deciding.
